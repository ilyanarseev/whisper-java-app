# whisper-java-app
Приложение для записи голоса с микрофона и преобразования его в текст с помощью Whisper JNI.

## Возможности
- Запись звука с микрофона в течение 10 секунд
- Распознавание речи в реальном времени с отображением текста по мере говорения
- Сохранение результата в файл result.txt (кодировка UTF-8) в папке запуска
- Использование OpenSource библиотеки Whisper-java для транскрибации
- Графический интерфейс на Swing

## Требования
- Java 11 или новее
- Apache Maven 3.6.3
- Микрофон
- Модель Whisper (в исходном коде используется ggml-tiny.bin)
	- Скачать можно отсюда: https://huggingface.co/ggerganov/whisper.cpp/tree/main
	- Модель должна находиться в папке models в корне проекта

## Сборка и запуск проекта
```
# Клонирование репозитория
git clone https://github.com/ilyanarseev/whisper-java-app.git
cd whisper-java-app

# Сборка проекта
mvn clean package

# Запуск проекта
java -jar target/whisper-java-app-1.0.0-jar-with-dependencies.jar
```

## Работа с программой
1. Запустите программу — откроется окно с кнопкой "Начать запись (10 сек)"
2. Нажмите кнопку — начнётся 10-секундная запись, на кнопке будет отображаться таймер
3. Говорите в микрофон — распознанный текст будет появляться в текстовом поле по мере говорения (каждые 2 секунды)
4. После окончания записи:
	- Финальный текст отобразится в окне
	- Результат сохранится в файл result.txt в папке запуска программы
	- Кнопка снова станет активной для новой записи
